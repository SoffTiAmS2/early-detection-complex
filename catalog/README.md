# Module Catalog

`catalog/honeypots.json` - tracked registry будущих honeypot-модулей.

Модуль считается рабочим только если у него есть:

- реальная установка;
- config contract;
- event adapter;
- health check;
- smoke test;
- список поддержанных архитектур;
- понятный resource class.

Пункт в UI без этих вещей не считается поддержкой.
