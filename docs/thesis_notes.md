# Thesis Notes

Проект используется как практическая база ВКР по распределенному комплексу honeypot-сенсоров.

## Что уже реализовано в каркасе

- центральный HTTP-узел приема событий;
- генератор конфигураций сенсоров;
- развертывание настоящего Cowrie-контейнера;
- агент отправки логов;
- агент статуса;
- скрипт установки центра.

## Что важно подчеркнуть в ВКР

- текущая реализация работает с Cowrie как реальным open-source honeypot;
- архитектура оставляет место для OpenCanary/Conpot, но они должны добавляться только с реальным контейнером и генератором конфига;
- центральный узел отделен от сенсоров, поэтому платы остаются легкими;
- центр ставит Docker, конфигурацию и compose на сенсор по SSH.
